## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Development Approach for NestJS projects with Mongodb

### Create a database module

1. Create a separate module for database
2. create a database providers file having async factorymethod for creating connection to the db as well as configuration options.
3. Also, in providers, create model with the connection
4. Return the connection
5. In the module, export the created provider
6. Import database module in every subsequent module which needs it

### Module wise checklist

1. Create Schema(include virtuals if needed)
2. Create DTO with JoiValidation for each type
3. Create JoiValidation Custom pipe
4. Validate controller input using Custom joi validation pipe on DTO
5. Create Provider, where model is initialised as the factory method. Give a provider token name(initialsed as a constant). Inject Database provider token name(initialise in a constant) in each provider model
6. Initialise Module with importing database module and adding above mentioned provider in provider array
7. In Service, inject the provider token for the model in the constructor with @Inject decorator
8. Optionally, implement Middleware to intercept the request response cycle. Consume in the module class
